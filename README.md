# burnet
Extract the version of Xcode built from the IPA file.  
Inspired by [here](https://qiita.com/bricklife/items/8b7c9dc4f7ab164e738d).
  
## How to use
  
```bash
> python main.py {IPA_FILE_PATH}
12A7300 # Xcode 12.0.1 (12A7300)
```
  
## Python Version
  
```bash
> python -V
Python 3.8.5
```
